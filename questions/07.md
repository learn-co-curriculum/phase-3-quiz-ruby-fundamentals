# Fundamentals: Operators

What would the following statement return?

```rb
letter = "b"

if letter == "a" || letter == "e" || letter == "i" || letter == "o" || letter == "u"
  "vowel"
else
  "not a vowel"
end
```

## Correct

"not a vowel"

> Right! The `||` (OR) operators for the `if` condition will all evaluate to `false`.
