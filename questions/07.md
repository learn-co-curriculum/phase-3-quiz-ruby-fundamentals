# Fundamentals: Conditionals

Which of the following is a case statement?

## Incorrect

```
name = "Steven"

if name == "Steven"
  "Hi, #{name}"
else
  puts "Hi, stranger!"
```

## Correct

```
name = "Steven"

case name
when "Steven"
  puts "Hi, #{name}"
when "Amanda"
  puts "Welcome back, #{name}"
when "Admin"
  puts "You have all the power!"
end
```

## Incorrect

```
def case
  puts "Am I a case statement?"
end

```
