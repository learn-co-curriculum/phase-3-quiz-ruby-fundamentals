# Fundamentals: Conditionals

Of the following two examples, which example uses the case statement the best?

## Incorrect

```
name = "Steven"

case name
when "Steven"
  puts "Hi, #{name}"
when "Amanda"
  puts "Welcome back, #{name}"
when "Admin"
  puts "You have all the power!"
end
```

## Incorrect

```
grade = 95

case grade
when 90..100
  "A" 
when 80..90
  "B"
when 70..80
  "C"
when 60..70
  "D"
when 0..60
  "F"
end
```

## Correct

Both are good as case statements

## Incorrect

I don't know.
