# Fundamentals: Looping

Here is a repeated list of&nbsp;`puts`&nbsp;statements.

```
puts "hi"
puts "hi"
puts "hi"
puts "hi"
puts "hi"
```

How would you repeat this with `loop`/`break` 100 times?

## Correct

```
counter = 0

loop do
  puts "hi"
  counter += 1
  break if counter == 100
end
```

## Incorrect

```
loop do
  puts "hi"
  break if counter == 100
end
```

## Incorrect

```
loop do
  puts "hi"
end
```
